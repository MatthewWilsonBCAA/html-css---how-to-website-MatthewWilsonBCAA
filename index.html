How to Skyrim

Chapter 1: Early Game

Step 1: Boot the Game

This step is pretty simple. Just put the disc in (if you have a physical copy), click on it, and let it load to the main menu.

Step 2: Start the Game

This step is also pretty easy. Just navigate to the option "New Game" and click it.

Step 3: Learn the Controls

The controls to this game can get tricky, so be sure to use Helgen as a training spot to learn them!
